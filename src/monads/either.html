The Either monad.

module.exports = Either = derive abstract-either, do

Group: Applicative

of type b -> Either(a, b)

Creates a new Either monad with a Right value.

of: (v) -> derive Either, { value: v, isRight: true }

of type @Either(a, b -> c), f:Applicative => f(b) -> f(c)

Applies a function from inside of the Either monad to another Applicative type.

ap: (b) -> @chain (f) -> b.map f

Group: Functor

of type @Either(a, b) -> (b -> c) -> Either(a, c)

Transforms the value of the Either monad using a regular function.

map: (f) -> @chain (v) ~> @of (f v)

Group: Monad

of type @Either(a, b) => (b -> Either(a, c)) -> Either(a, c)

Transforms the value of the Either monad using a function to a monad of the same type.

chain: (f) -> @fold do * (l) ~> @Left l * (r) -> f r

Group: Recoverable

of type @Either(a, b) => (a -> Either(c, b)) -> Either(c, b)

Applies a function to the Left side of the Either monad.

or-else: (f) -> @fold do * (l) -> f l * (r) ~> @Right r

Group: Show

of type Unit -> String

Returns a textual representation of the Either monad.

to-string: -> @fold do * (l) -> "Either.Left(#l)" * (r) -> "Either.Right(#r)"

Group: Eq

of type @Either(a) => Either(a) -> Boolean

Tests if an Either monad is equal to another Either monad.

is-equal: (a) -> @is-left is a.is-left and \ @is-right is a.is-right and \ @value is a.value

Ungrouped

of type @Either(a, b) => (a -> c) -> (b -> c) -> c

Catamorphism for the disjunction.

fold: (f, g) --> | @isLeft => f @value | @isRight => g @value

of type @Either(a, b) => Unit -> Either(b, a)

Swaps the disjunction values.

swap: -> @fold do * (l) ~> @Right l * (r) ~> @Left r

of type @Either(a, b) => (a -> c) -> (b -> d) -> Either(c, d)

Maps both sides of the disjunction.

bimap: (f, g) -> @fold do * (l) ~> @Left (f l) * (r) ~> @Right (g r) # The Left case of the ADT. Left = derive Either, do isLeft: true
